# Padrões Comportamentais

Bem-vindo à documentação dos Padrões Comportamentais! Aqui você encontrará explicações e demonstrações de código para diversos padrões de design comportamental.

## Padrões Disponíveis

- **Strategy**: Permite trocar algoritmos em tempo de execução.
- **Observer**: Notifica mudanças para múltiplos objetos.
- **Iterator**: Acesso sequencial aos elementos de uma coleção.
- **Command**: Encapsula solicitações como objetos.
- **State**: Altera comportamento baseado no estado interno.
- **Template Method**: Define esqueleto de algoritmo com passos customizáveis.
- **Chain of Responsibility**: Passa solicitações por uma cadeia de handlers.

Clique nos links na barra lateral para explorar cada padrão com exemplos práticos baseados nos commits do projeto.
